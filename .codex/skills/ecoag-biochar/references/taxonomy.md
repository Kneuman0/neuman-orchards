## 2. Question Routing Patterns

- If the user asks "What is biochar, how is it made, what feedstocks or pyrolysis conditions matter?"
  Route to `Vera-Ruiz-et-al-2020-use-of-biochar-in-agriculture.pdf`.

- If the user asks "What does biochar generally do to soil fertility, pH, water retention, nutrient holding, plant growth, or heavy-metal risk?"
  Start with `Vera-Ruiz-et-al-2020-use-of-biochar-in-agriculture.pdf`.

- If the user asks "Should biochar be charged, aged, or composted before use?" or "Is co-composted biochar better than raw biochar?"
  Start with `Kammann-et-al-2015-co-composted-biochar.pdf`, then use `Vera-Ruiz-et-al-2020-use-of-biochar-in-agriculture.pdf` for broader background.

- If the user asks "Can biochar reduce nitrate loss, improve nutrient retention, or make compost more effective?"
  Route first to `Kammann-et-al-2015-co-composted-biochar.pdf`.

- If the user asks "Can biochar suppress disease?" or "Can biochar help with Fusarium, root rot, or tomato soilborne disease?"
  Route first to `Jaiswal-et-al-2017-biochar-tomato-disease-suppression.pdf`.

- If the user asks "How might biochar affect the soil microbiome, beneficial microbes, or disease-suppressive biology?"
  Route first to `Jaiswal-et-al-2017-biochar-tomato-disease-suppression.pdf`.

- If the user asks "Will biochar help under drought, water stress, or deficit irrigation?" or "Will biochar improve cucumber yield in dry conditions?"
  Route first to `Singh-et-al-2025-cucumber-biochar-deficit-irrigation.pdf`, then use `Vera-Ruiz-et-al-2020-use-of-biochar-in-agriculture.pdf` for broader water-retention mechanisms.

- If the user asks "What application rates are supported locally?"
  Check `Singh-et-al-2025-cucumber-biochar-deficit-irrigation.pdf` for explicit field rates in cucumber and `Vera-Ruiz-et-al-2020-use-of-biochar-in-agriculture.pdf` for general review context, but answer cautiously because this folder does not contain a robust rate-comparison synthesis across crops and soils.

## 5. Unresolved Gaps / Weak Evidence In This Folder

- Application-rate guidance is weak outside the cucumber paper; local sources do not provide a stable cross-crop prescription.
- Disease suppression evidence is strong locally for tomato/Fusarium but not broad enough to generalize confidently to other crops or pathogens.
- Water-management evidence is mixed: the cucumber paper shows hydraulic improvement but only minimal short-term growth/yield benefit from biochar itself.
- Feedstock and pyrolysis-condition dependence is repeatedly acknowledged, but this folder does not contain a dedicated side-by-side local comparison study for multiple feedstocks under the same field conditions.
- Long-term field performance is underdeveloped here; this set is stronger on mechanisms and targeted cases than on multi-year farm-scale outcomes.
