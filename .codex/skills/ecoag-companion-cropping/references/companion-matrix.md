Purpose: give the `ecoag-companion-cropping` skill a broad, retrieval-friendly matrix of useful garden plant pairings that are relevant to this zone-7 workspace. This is not a folklore-only "plant these two together" chart. It separates stronger evidence from weaker guild logic and names the mechanism wherever possible.

## How To Read This

Evidence tier:

- `A` = directly supported by the local review set or by strong extension / peer-reviewed field evidence.
- `B` = reasonable agronomic logic with some supporting evidence, but still context-dependent.
- `C` = mostly traditional or guild logic; useful candidate, but weak direct evidence.

Mechanism tags:

- `Trap` = attracts pests away from the focal crop.
- `Disrupt` = makes host-finding or colonization harder.
- `Enemy support` = feeds or shelters predators / parasitoids.
- `N transfer` = legume-to-non-legume nitrogen transfer is possible, but conditional.
- `Niche stacking` = different canopy, rooting, or timing reduces direct competition.

Important caution:

- The local literature is strongest for trap cropping, aphid management, maize-bean-squash systems, brassica intercrops, and conditional legume-to-non-legume nitrogen transfer.
- The local literature is weaker for broad folklore claims like "improves flavor" or "all scented herbs repel pests."

## Focal Crop Matrix

|---|---|---|---|---|
| Corn / flint corn / sweet corn | Pole bean, winter squash or pumpkin | Classic milpa logic: corn provides vertical structure, beans occupy the climbing niche, squash covers soil and suppresses open-ground weed pressure while diversifying the canopy. | A | Best when spacing and timing keep corn from being smothered early. Beans do not automatically "feed" corn quickly; N transfer is real but conditional and often delayed. [L4][L6] |
| Corn / flint corn / sweet corn | Crimson clover, common vetch, hairy vetch | Legume intercrops can contribute biologically fixed N over time and improve whole-system diversity. | B | Treat this as conditional fertility support, not a guarantee of in-season corn feeding. Root contact, density, soil N, water, and stage all matter. [L4][L5] |
| Pole bean | Corn, sunflower | Good physical trellis and vertical niche stacking. | B | This is mostly structural logic unless managed as a true intercrop. Sunflower is useful only if it does not over-shade the beans. [L4] |
| Pole bean | Squash / pumpkin | Shared three-sisters style ground occupation: bean climbs, squash shades soil. | B | Works best where bean vigor does not bury young squash and harvest access is planned. [L4][L6] |
| Bush bean | Radish, lettuce | Different rooting depth and harvest timing can improve bed occupancy and reduce bare soil periods. | C | Mostly spatial logic; direct pest-suppression evidence is weak here. |

| Tomato | Basil | Shared heat-loving habit and common garden management needs. Traditional claim for pest disruption is plausible but weakly proven in open-field gardens. | C | Keep as a convenience and pollinator herb pairing, not as a guaranteed pest-control strategy. |

| Potato | Soybean, sorghum, winter wheat | Local aphid review cites these as trap plants that reduced potato virus Y incidence in seed potato by affecting aphid vectors. | A | This is a real vector-management use, but it is a field-style strategy, not a small mixed-bed silver bullet. [L1] |
| Potato | French marigold (`Tagetes patula`) | Best external support is for nematode suppression, not broad beetle repellency. | B | Works better as pre-plant conditioning or dense companion strips than as a token flower beside one potato hill. [E3] |

## Fast Rules For Retrieval

- If the user wants the strongest pairings in this workspace, start with:
  - corn + bean + squash
  - cucurbits + Blue Hubbard perimeter trap crop
  - brassicas + clover / cilantro / alliums
  - carrots + onion-family crops
  - peppers + chives or other alliums for aphid disruption

- If the user asks whether legumes "feed" neighboring crops:
  - answer that transfer can happen, but is conditional and often not equivalent to a fertilizer application. [L5]

- If the user asks for "best companions" for a crop with no clear mechanism:
  - prefer insectary or niche-stacking plants from the local inventories over folklore pairings.

- If the user asks for perennial guild plants:
  - prioritize compact alliums plus long-bloom herbs like thyme, oregano, anise hyssop, bee balm, sweet cicely, or lovage, and mark those as mostly `B` or `C` unless there is direct evidence.

## Things This Matrix Deliberately Avoids Overclaiming

- Basil does not get treated here as a universally proven tomato protector.
- Marigold is not treated as a universal pest repellent; the strongest support is narrow and nematode-related.
- Legumes are not treated as instant nitrogen pumps for neighboring crops.
- "Good companion because grandma said so" claims are not promoted above evidence-backed trap, disruption, enemy-support, or niche-stacking mechanisms.
